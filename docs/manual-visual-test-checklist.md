# Manual Visual Test Checklist

- Open `http://localhost:3000`.
- Confirm the dashboard loads.
- Confirm the project list is visible.
- Create a test project.
- Open the project editor.
- Change branding colours.
- Select a template.
- Run Recon, Security, Verify, and Backup agents.
- Create a backup.
- Run a build.
- Open the generated preview path.
- Confirm the preview reflects the business name, tagline, selected template, and branding colours.
- Test mobile width manually.
- Capture screenshots if possible.
